# Responsive Navigation Bar

## 📌 Project Overview

This project is a modern and fully responsive navigation bar developed using **HTML, CSS, and JavaScript**. It provides smooth navigation across desktop, tablet, and mobile devices with an interactive user interface and attractive animated background.

## 🎯 Objective

To create a responsive navigation bar that includes:

- Logo and menu items
- Hamburger menu for mobile devices
- JavaScript toggle functionality
- Smooth navigation across different screen sizes

## ✨ Features

- Responsive Navigation Bar
- Mobile-Friendly Hamburger Menu
- Interactive Navigation Links
- Get Started Button with Popup Message
- Animated Gradient Background
- Floating Glow Effects
- Glassmorphism Navbar Design
- Smooth Hover Animations
- Cross-Device Compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
